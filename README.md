# Численные методы
В данном проекте реализованы различные численные методы, применяемые для решения широкого спектра математических задач.
## 1 задание
### цель работы
Реализация и исследование численных методов для вычисления элементарных и сложных функций. Оценка вычислительной погрешности и сравнение результатов с точными значениями, полученными с помощью встроенных функций Python.
### реализованные методы
* Метод Ньютона (касательных) для вычисления квадратного корня
* Разложение в ряд Тейлора
* Композиция функций
## 2 задание
### цель работы
Сравнительный анализ прямых и итерационных методов решения СЛАУ. Исследование их точности, скорости сходимости и применимости к системам с различными свойствами матриц.
### реализованные методы
* LU-разложение с частичным выбором ведущего элемента
* QR-разложение методом Хаусхолдера
* Метод простых итераций
* Метод Зейделя
## 3 задание
### цель работы
Нахождение корня уравнения x² + 1 - arccos(x) = 0 комбинированным методом. Найти решение системы уравнений cos(y) + x = 1.5 и 2y - sin(x-0.5) = 1.
### реализованные методы
* Локализация корня на интервале [-1, 1]
* Комбинация метода Ньютона и половинного деления
* Графический метод для начального приближения
* Метод Ньютона для систем
* Метод продолжения по параметру (λ-метод)
## 4 задание
### цель работы
Аппроксимация функции f(x) = x² + 1 - arccos(x) на интервале [-0.8, 0.8] с анализом влияния выбора узлов на точность. Сравнение эффективности сплайнов разной степени (линейных, квадратичных, кубических) с интерполяционным полиномом Ньютона.
### реализованные методы
* Интерполяционные полиномы Лагранжа и Ньютона
* Равномерные и чебышёвские узлы интерполяции
* Линейные сплайны (S₁,₀)

* Квадратичные сплайны (S₂,₁)

* Кубические сплайны (S₃,₂)
## 5 задание
### цель работы
Нахождение собственных значений и векторов матрицы различными методами.
### реализованные методы
* Степенной метод (максимальное по модулю собственное значение)

* Обратный степенной метод (все собственные значения со сдвигами)

* QR-алгоритм (все собственные значения)

## 6 задание
### цель работы
Аппроксимация зашумленных данных полиномами различной степени.
### реализованные методы
* Решение нормальных уравнений

* Ортогональные полиномы
